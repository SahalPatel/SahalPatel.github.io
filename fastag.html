Project Overview

FASTag generates large volumes of transaction logs, containing metadata such as vehicle class, time of toll crossing, GPS zone, lane information, and payment status. By applying machine learning to these logs, the system detects unusual patterns that deviate from normal behavior. This helps toll agencies reduce revenue losses and strengthen road security systems.

Dataset Description

The dataset includes:

  • Transaction ID
  • Vehicle ID / Tag ID
  • Toll booth and lane
  • Timestamp
  • Transaction amount
  • Vehicle class
  • Account balance
  • Route-based metadata

Missing values are cleaned, timestamp features are extracted, and categorical attributes (such as vehicle class or lane ID) are encoded before model training.

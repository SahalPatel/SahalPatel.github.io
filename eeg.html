EEG Topographic Visualization Using MNE-Python

This project focuses on generating EEG topographic (scalp) maps using power spectral density (PSD) data and MNE-Python. The goal is to visualize how brainwave activity (such as alpha, beta, gamma bands) is distributed across different regions of the scalp.

Project Overview

EEG signals provide valuable information about cognitive, neurological, and behavioral processes. This project loads PSD values for multiple electrodes, maps them to their spatial coordinates using a standard EEG montage, and visualizes the data as 2D heatmaps (topomaps). These topomaps make it easy to identify patterns such as increased alpha activity in the occipital region or elevated gamma activity during tasks.

Dataset Description

The dataset consists of:

  • PSD values for different frequency bands (alpha, beta, gamma, etc.)
  • Channel names such as Fp1, Fp2, Cz, O1, O2
  • Montage coordinates using a standard EEG cap layout (like 10-20 system)

The PSD values are loaded from CSV files and preprocessed using pandas and numpy. The electrode coordinates are handled by MNE using built-in montages.

Signal Processing Workflow

The processing pipeline includes:

  • Loading PSD data from CSV
  • Merging PSD values with electrode coordinates
  • Normalizing or scaling the power values
  • Generating topomaps using mne.viz.plot_topomap
  • Visualizing band-power distribution for alpha, beta, and gamma bands

MNE-Python handles interpolation, contour generation, and color mapping to produce clean and meaningful visualizations.
